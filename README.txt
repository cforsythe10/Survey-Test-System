Drexel University Software Architecture
HW4-Full survey test system

This is a fill java application to create, save, load, and take surveys and tests using several
OOP design patterns. This application is designed to work from the console, with prompts.
It is easiest to use this program by placing the entire folder for the system in an ide and
running it, though command line compilation will work as well.

Essays do not allow multiple responses
I used serialization for saving/loading
The sample files are in the working directory, one step above the src folder
